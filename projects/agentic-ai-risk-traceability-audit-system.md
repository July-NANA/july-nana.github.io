# Agentic AI Risk Traceability Audit System

## Problem
Large multi-agent systems create long chains of tool calls and interactions, which makes accountability, identity verification, and risk tracing difficult in regulated or high-stakes environments.

## Data
The system processed agent interaction events, identity records, call traces, and workflow metadata generated by a large-scale multi-agent experimental platform supporting 100-level model interaction.

## Approach
I combined a LangGraph-based multi-agent platform with backend service orchestration, GraphQL query interfaces, and blockchain smart contracts on Sui and Ethereum to register identities, record interaction events, and construct auditable trajectories.

## Outcome/Impact
The project delivered an end-to-end audit and traceability mechanism for AI-agent interactions and improved traceability efficiency from O(n) to O(1), making verification and security review substantially faster.
