# Wafer Defect Monitoring Application

## Problem
Semiconductor manufacturing workflows required more efficient defect inspection and visualization so engineering teams could reduce manual screening effort and accelerate yield monitoring.

## Data
The application used wafer inspection data, defect imagery, and processed image outputs stored across database records and object storage to support monitoring and visual review.

## Approach
I developed backend services with Spring Boot and MySQL, connected them with S3-based storage, and supported image processing workflows using KNN, interpolation, and related methods to redraw and analyze wafer images.

## Outcome/Impact
The project increased defect detection efficiency by 90% and reduced manual screening cost by improving visualization, query speed, and backend support for large inspection datasets.

I was responsible for Java development, backend query optimization, database design, and image-algorithm-related implementation support.
